## About

**BuyHigh** is a web platform that provides **free AI analysis reports** for Korean stocks.
It turns complex charts and financial metrics into reports anyone can understand.

## Features

- **Chart Analysis** — AI interprets candles, moving averages, and volume into plain-language reports with actionable guidance
- **Valuation Analysis** — Value assessment and risk factors based on key metrics like PER, PBR, and ROE
- **Stock Recommendations** — Data-driven trade ideas with position and range insights
- **AI Reports** — Stock analysis auto-generated by multiple LLMs (GPT · Gemini)
- **Free Analysis Feed** — Daily-updated lists of free chart and valuation analyses

## Tech Stack

| Area | Technologies |
|------|-----------|
| Backend | Java 21, Spring Boot 3.5, Spring WebFlux, Spring Security |
